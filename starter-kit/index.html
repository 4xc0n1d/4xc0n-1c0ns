Usage as a Font

Import The Css

Copy-paste the stylesheet link into your head to load our CSS 

<link rel="stylesheet" type="text/css" href="https://www.axcon.my.id/assets/docs/fonts/axcon.css">

HTML

To use an icon on your page, add a prefix ax- 

                    <i class="ax-weather"></i>
                    <i class="ax-youtube"></i>
Using The SVG Code

Import The JS

Copy-paste the link into your Script to load our JS 

                    <link rel="stylesheet" type="text/css" href="https://www.axcon.my.id/assets/docs/svg/axcon/axcon.css">
                    <script defer src="https://www.axcon.my.id/assets/docs/svg/axcon.js"><script>

HTML

To use an icon on your page, add a prefix ax- 

                    <svg> class="ax ax-rewind">
                      <use> xlink:href="#ax-rewind"></use>
                    </svg>
